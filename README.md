# 你好，我是 `zgx197`

欢迎来到我的 GitHub 主页。

这个仓库会作为我的个人 Profile README，集中展示我正在关注的方向、持续维护的项目，以及我对开源协作的理解与实践。

## 关于这里

我更喜欢把代码仓库做成可以长期维护、方便他人阅读和参与的样子：

- 有明确的问题边界
- 有可复现的使用方式
- 有持续补充的文档说明
- 有可以逐步迭代的工程结构

## 我关注的方向

- 实用型开源工具
- 自动化与效率提升
- 工程化实践与开发体验
- 让项目更易读、更易维护、更容易协作

## 近期计划

- 持续整理并公开个人项目
- 为已有仓库补充更完整的文档与示例
- 把零散想法沉淀成可复用的开源成果

## 开源协作方式

如果你对某个项目感兴趣，欢迎通过下面的方式参与交流：

- 提交 Issue 反馈问题或提出建议
- 提交 Pull Request 一起完善实现
- 在项目页面留下 Star，帮助我判断哪些方向更值得继续投入
