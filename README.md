## 关于这里

这里主要记录我正在推进的开源项目，以及一些持续打磨中的工具、工作流和游戏原型。

相比一次性完成的 Demo，我更在意项目是否能被长期维护、被他人快速理解，并在后续迭代中持续生长。

## 我关注的方向

- AI 工具与 Agent 工作流
- 游戏原型、玩法验证与运行时能力
- 从设计到交付的工程化流程
- 文档、约定与可复用结构的持续沉淀

## Current Focus

### [resume-interview-workbench](https://github.com/zgx197/resume-interview-workbench)

本地 AI 面试工作台，面向结构化简历、岗位模板与可观测多轮追问。

当前里程碑：面试链路与结果可观测性

近期迭代目标：
1. 把岗位模板配置收敛成固定格式。
2. 给多轮追问流程补上可视化日志。
3. 统一面试总结和评分结果的输出结构。
